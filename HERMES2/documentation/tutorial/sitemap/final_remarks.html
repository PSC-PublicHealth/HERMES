Final HERMES Remarks

This tutorial aimed to cover both the running of a baseline simulation, and various experimental procedures that can be done to "tweak" the model scenario.

The example country that has been used so far has been the Niger model, due to the relative completeness of the data. It is, however, an idiosyncratic model - there are a few things (e.g. "open vial waste") that doesn't have to be dealt with in the other country models.

Therefore, if you have any questions about the particular quirks of the model, please let one of us know!

Remember: while HERMES appears pretty complex, by practicing different scenarios, you can become proficient at HERMES!


Final HERMES Country Model Notes

The countries that we currently have models for are listed below. A sketch of each model's idiosyncrasies are after the dash:

Current Country Models:

  • Niger - the first model constructed, it has an idiosyncratic ordering policy & a need for an open vial waste file (which isn't needed in the other models)
  • Thailand - the second model constructed, its ordering policy has shifted from a pre-VMI (vendor-managed inventory) system to a VMI system
    • Note: from here, all the HERMES models are using the model_generic.py file instead of a country-specific Python file.
  • Vietnam - the third model constructed, it uses "catch-all bins" and "surrogates" to model the whole country's supply chains based on the explicit data of a few locations throughout the country
  • Senegal - the fourth model constructed, it again uses "catch-all bins" and "surrogates"
  • Chad - the fifth model constructed, there are two versions of this model: one using the "current" Chad supply chain model, and one that uses a "Revitalized" Chad model (which seeks to optimize the supply chain)
  • Kenya - this model is, by far, the largest model constructed, with over 4,000 locations scattered throughout the country
  • Benin - a draft of the model was created throughout the summer of 2012; the model was used throughout September 2012

Note: other country models are being discussed and/or are in the works, but no definitive input files have been created yet.
